Welcome to the Zodiac Sign Finder!

This project allows users to discover their zodiac sign based on their birth date. It supports both English and Brazilian Portuguese languages.

Features

  • Enter your birth date to find your zodiac sign.
  • Available in both English and Portuguese.
  • Responsive design for a seamless experience on any device.

How to Use

  1. Visit the Home Page to enter your birth date.
  2. Select your preferred language from the dropdown menu.
  3. Click "Consultar Signo" to see your zodiac sign and description.
  4. Review the result and learn more about your zodiac sign!

Project Structure

The project consists of the following files:

  • index.php: The home page where users input their birth date and select a language.
  • show_zodiac_sign.php: The result page that displays the zodiac sign and description.
  • signos.xml: The XML file containing zodiac sign information in both English and Portuguese.
  • styles.css: The global CSS file for styling the pages.
  • README.html: This file, providing an overview of the project.
